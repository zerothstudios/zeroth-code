# Zeroth Studios — Code Exercises

Hands-on code exercises for [Zeroth Studios](https://zerothstudios.com) podcasts. Each episode has exercises that let you see the concepts in action.

**You don't need any programming experience.** We'll walk you through everything.

---

## How to Run These Exercises

You have two options. Pick whichever sounds easier.

### Option A: Run in your browser (no installation)

This is the easiest way. You don't need to download or install anything.

1. Go to [Google Colab](https://colab.research.google.com) — it's a free tool from Google that lets you run code in your browser.
2. Sign in with any Google account (the same one you use for Gmail or YouTube works fine).
3. Click **File > Open notebook > GitHub**.
4. Type `zerothstudios` in the search box and hit Enter.
5. Find the exercise you want and click it.
6. To run the code, click the **play button** (the triangle icon) next to each code block, or press **Shift + Enter** on your keyboard.

That's it. The code runs right there in your browser. You can change things, re-run, and experiment without breaking anything.

### Option B: Run on your computer

This takes a few minutes to set up, but then you can run exercises offline anytime.

**Step 1 — Install Python** (skip this if you already have it)

- **Mac:** Python is usually already installed. Open the Terminal app (search for "Terminal" in Spotlight) and type `python3 --version`. If you see a version number, you're good. If not, download Python from [python.org/downloads](https://www.python.org/downloads/).
- **Windows:** Download Python from [python.org/downloads](https://www.python.org/downloads/). During installation, **check the box that says "Add Python to PATH"** — this is important.

**Step 2 — Download the exercises**

Click the green **Code** button at the top of this page, then **Download ZIP**. Unzip it anywhere on your computer.

**Step 3 — Open a terminal**

- **Mac:** Open the Terminal app (search "Terminal" in Spotlight, or find it in Applications > Utilities).
- **Windows:** Press the Windows key, type `cmd`, and open Command Prompt.

**Step 4 — Navigate to the exercises**

Type this in your terminal (change the path to wherever you unzipped the files):

```
cd Downloads/zeroth-code-main
```

**Step 5 — Run an exercise**

```
python3 zeroth-ai/season-1/ep01-what-makes-something-smart/exercises.py
```

On Windows, use `python` instead of `python3`:

```
python zeroth-ai/season-1/ep01-what-makes-something-smart/exercises.py
```

You'll see the output right in your terminal. To try a different exercise, change the file path.
