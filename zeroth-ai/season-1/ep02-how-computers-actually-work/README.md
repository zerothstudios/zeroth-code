# Episode 2: How Computers Actually Work

Companion exercises for **Zeroth: AI** Episode 2.

**The big idea:** Every computer runs on binary (0s and 1s) and logic gates. AI isn't magic hardware — it's the same transistors, same gates, same fetch-decode-execute cycle.

## Listen First

These exercises make more sense after you've listened to the episode.

- [Episode 2 page](https://zerothstudios.com/zeroth-ai/episode-2-how-computers-actually-work/)

## What You'll See

1. **Binary Converter** — Convert decimal numbers to binary. See how 255 = 11111111.
2. **Bits and Bytes** — How adding one bit doubles the possibilities. Exponential growth in action.
3. **Logic Gates** — NOT, AND, OR. The three operations that everything is built from.
4. **Half-Adder** — Build addition from scratch using only logic gates. No math — just AND, OR, NOT.
5. **ASCII** — Every letter is a number. The word "AI" in binary is 01000001 01001001.

## How to Run

### Option A: Run in your browser (easiest)

1. Go to [Google Colab](https://colab.research.google.com).
2. Sign in with any Google account (Gmail, YouTube, etc.).
3. Click **File > Open notebook > GitHub**.
4. Type `zerothstudios` in the search box and hit Enter.
5. Find `zeroth-ai/season-1/ep02-how-computers-actually-work/exercises.py` and click it.
6. Click the **play button** next to each code block, or press **Shift + Enter**.

### Option B: Run on your computer

**Install Python** (skip if you already have it):
- **Mac:** Open Terminal and type `python3 --version`. If you see a number, you're good. Otherwise, download from [python.org/downloads](https://www.python.org/downloads/).
- **Windows:** Download from [python.org/downloads](https://www.python.org/downloads/). **Check "Add Python to PATH"** during install.

**Download and run:**
1. Download this repo: click the green **Code** button at the [top of the repo](https://github.com/zerothstudios/zeroth-code), then **Download ZIP**. Unzip it.
2. Open a terminal (Mac: Terminal app; Windows: Command Prompt).
3. Navigate to the unzipped folder:
   ```
   cd Downloads/zeroth-code-main
   ```
4. Run the exercises:
   ```
   python3 zeroth-ai/season-1/ep02-how-computers-actually-work/exercises.py
   ```
   On Windows, use `python` instead of `python3`.
