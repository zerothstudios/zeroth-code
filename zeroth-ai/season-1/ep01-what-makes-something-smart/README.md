# Episode 1: What Makes Something Smart?

Companion exercises for **Zeroth: AI** Episode 1.

**The big idea:** AI is a machine that figures out rules from experience, instead of being told the rules by a programmer.

## Listen First

These exercises make more sense after you've listened to the episode.

- [Zeroth: AI show page](https://zerothstudios.com/zeroth-ai/)
- [Episode 1 page](https://zerothstudios.com/zeroth-ai/episode-1-what-makes-something-smart/)
- [Episode 1 blog post](https://zerothstudios.com/blog/what-makes-something-smart/)

## What You'll See

1. **The Thermostat** — A rule-based system. No learning, no adaptation. It follows one rule a human wrote.
2. **Rule-Based Spam Filter** — The old way. Programmers wrote word lists. Spammers changed words. Arms race.
3. **Learning Spam Filter** — The AI way. The system learns patterns from examples. Nobody programmed a word list.
4. **The Intelligence Spectrum** — From thermostats to three-year-olds. Intelligence isn't one thing.

## How to Run

### Option A: Run in your browser (easiest)

Open the notebook in Google Colab — no installation needed:

1. Go to [Google Colab](https://colab.research.google.com).
2. Sign in with any Google account (Gmail, YouTube, etc.).
3. Click **File > Open notebook > GitHub**.
4. Type `zerothstudios` in the search box and hit Enter.
5. Find **`exercises.ipynb`** in the `ep01-what-makes-something-smart` folder and click it.
6. Click the **play button** (▶) next to each code block, or press **Shift + Enter**.

Each exercise has an explanation above it and a code block you can run and modify.

### Option B: Run on your computer

**Install Python** (skip if you already have it):
- **Mac:** Open Terminal and type `python3 --version`. If you see a number, you're good. Otherwise, download from [python.org/downloads](https://www.python.org/downloads/).
- **Windows:** Download from [python.org/downloads](https://www.python.org/downloads/). **Check "Add Python to PATH"** during install.

**Download and run:**
1. Download this repo: click the green **Code** button at the [top of the repo](https://github.com/zerothstudios/zeroth-code), then **Download ZIP**. Unzip it.
2. Open a terminal (Mac: Terminal app; Windows: Command Prompt).
3. Navigate to the unzipped folder:
   ```
   cd Downloads/zeroth-code-main
   ```
4. Run the exercises:
   ```
   python3 zeroth-ai/season-1/ep01-what-makes-something-smart/exercises.py
   ```
   On Windows, use `python` instead of `python3`.
